- **Headlines:** 2-12 words (punchy for teens/rentals, descriptive for young adults)
- **Subheadlines:** 1 sentence (12-20 words), addresses main barrier
- **Body copy:** 30-180 words depending on section
- **Testimonials:** 2-3 sentences (50-80 words each)

- **High image:** Identity Grid (30:70), Kids (25:75), Teens (30:70)

- **Hard CTAs:** Class exploration buttons, rental booking buttons
- **Soft CTAs:** "Ready to explore?" links, testimonial follow-ups
- **Routes:** All demographic sections route to /kids, /teens, /young-adults, /adults, /active-adults, /rentals landing pages
