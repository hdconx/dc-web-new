## DESIGN DIRECTION RECOMMENDATIONS (3 Options)

### Direction 1: "Clean Grid Router"
- Top: Simple 6-tile identity grid (text + icon or AI silhouette)
- Middle: Demographic sections with text + one image each (no captions)
- Bottom: Generic sections interspersed
- Style: Dark background, clean typography, whitespace-heavy, professional
- Feeling: "We serve everyone with equal respect"

### Direction 2: "Staggered Flow"
- Top: Angled/asymmetrical identity grid (more dynamic)
- Middle: Demographic vignettes on alternating sides (L-R-L-R)
- Bottom: Generic sections same pattern
- Style: Dark background, motion/asymmetry, feels more contemporary
- Feeling: "We're welcoming AND energetic"

### Direction 3: "Card Reveal"
- Top: Identity grid as cards (hover reveals class types)
- Middle: Demographic vignettes as cards (hover shows more detail)
- Bottom: Generic sections as card-style blocks
- Style: Dark background, interactive, cards fade-in on scroll
- Feeling: "Explore your demographic without overwhelm"
