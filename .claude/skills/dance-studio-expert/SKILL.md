## How to Apply This Expertise

### When Creating Marketing Plans:
1. Identify target segment (instructors? studios? content creators?)
2. Choose 2-3 channels from priority list
3. Create content calendar (90-day minimum)
4. Set metrics (bookings, inquiries, website traffic)
5. Budget allocation (70% proven channels, 30% experiments)

### When Optimizing Website:
1. Audit current conversion funnel
2. Identify friction points (too many steps? unclear pricing?)
3. A/B test one element at a time
4. Implement highest-impact changes first
5. Measure before/after conversion rates

### When Pricing New Services:
1. Research 3-5 competitors in area
2. Position based on amenities (premium if justified)
3. Create tiered options (good/better/best)
4. Test pricing for 30 days
5. Adjust based on booking rate (target 60-70% utilization)

### When Writing Content:
1. Know the audience pain points
2. Address objections before they arise
3. Show, don't just tell (use images/video)
4. Include clear CTAs every 2-3 paragraphs
5. Mobile-first (most dance community browses on phone)
