## REFERENCE: DESIGN ELEMENTS TO LOOK FOR

### Navigation Structure
- [ ] Top navigation by demographic (Kids | Teens | Adults | Seniors | Rentals)
- [ ] Clear age ranges (not vague "classes for all ages")
- [ ] Easy-to-find sections for each demographic
- [ ] No buried information

### Visual Diversity
- [ ] Multiple dance styles shown (not just one type)
- [ ] Different ages represented visually
- [ ] Professional but warm aesthetic
- [ ] Dark background or clean white space (not cluttered)

### Messaging
- [ ] Beginner-friendly language
- [ ] No gatekeeping ("elite," "advanced only")
- [ ] Warm, authentic tone
- [ ] Clear value prop for each demographic

### Layout Patterns
- [ ] Hero section doesn't alienate (shows diverse ages OR generic image)
- [ ] Demographic sections clearly separated (scrollable)
- [ ] Generic sections (philosophy, testimonials, studio space) work for everyone
- [ ] Rentals shown as important (if applicable)
